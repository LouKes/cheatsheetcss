Box Model: CSS styles

CSS properties

Property Description
Position The position property specifies the type of positioning method used for an element.
Float The float property is used for positioning and formatting content e.g. let an image float left to the text in a container.
Padding The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
Margins The CSS margin properties are used to create space around elements, outside of any defined borders.
Borders The CSS border properties allow you to specify the style, width, and color of an element's border.

CSS property values

Property Value Value description
static HTML elements are positioned static by default.
relative An element with position: relative; is positioned relative to its normal position.
fixed An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.
absolute An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).
sticky An element with position: sticky; is positioned based on the user's scroll position.
